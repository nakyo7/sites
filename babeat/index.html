  • 소진 완료 처리 시 재고 자동 차감
  • 이전 식단 불러오기로 빠른 식단 구성

자동 분류로 빠르고
간편한 큐브 등록

이름만 입력하면 카테고리와 아이콘을 자동으로 감지해 빠르게 등록할 수 있어요.

  • 이름 입력 시 카테고리·아이콘 자동 감지
  • 곡물, 고기, 채소 등 8가지 카테고리 분류
  • 슬라이더로 큐브 용량 간편 설정
  • "만들기 전" 큐브로 미리 식단 계획 가능

Features
이유식 관리, 이렇게 쉬워져요
매일 반복되는 이유식 준비, 바비트가 체계적으로 정리해 드려요.
🧊

큐브 재고 관리

곡물, 고기, 채소, 생선 등 카테고리별로 이유식 큐브를 등록하고 재고와 소비기한을 한눈에 확인하세요.

📅

주간 식단 계획

주간 캘린더에서 매일의 식단을 계획하고, 이전 식단을 불러와 빠르게 구성할 수 있어요.

📊

가용 재고 추적

식단에 배정된 큐브를 자동으로 계산해, 실제 사용 가능한 가용 재고를 정확히 알려드려요.

👶

다중 아기 지원

여러 아기의 이유식을 각각 관리하고, 아기별 프로필과 D-day를 추적하세요.

👨‍👩‍👧

가족 공유

초대 코드를 통해 파트너와 함께 아기의 이유식 정보를 실시간으로 공유하세요.

🔄

실시간 동기화

모든 데이터가 클라우드에 안전하게 저장되고, 여러 기기에서 실시간으로 동기화됩니다.
